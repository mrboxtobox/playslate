# 🎨 PlaySlate ✨

A magical drawing surface designed for kids to explore, create, and build amazing things! PlaySlate transforms the simple act of drawing into an interactive, engaging experience filled with sparkles, colors, and wonder.

## 🌟 Features

- **Custom Drawing Canvas**: Built from scratch using HTML5 Canvas API for smooth, responsive drawing
- **Kid-Friendly Interface**: Large, colorful buttons and intuitive controls designed for small hands
- **Magical Effects**: 
  - ✨ Sparkles that follow your cursor and appear on clicks
  - 🌈 Rainbow brush mode with animated colors
  - 🌟 Glow effects and magical animations
  - 🎭 Special magic wand tools
- **Touch Support**: Full support for tablets and touch devices
- **Color Palette**: 8 vibrant, kid-friendly colors
- **Brush Sizes**: Adjustable brush sizes from tiny to super thick
- **Clear Canvas**: Easy-to-use clear button with magical gradient background

## 🚀 Getting Started

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```
4. Open your browser and start creating magic!

## 🎮 How to Use

1. **Choose Your Colors**: Click on any of the colorful circle buttons to select your drawing color
2. **Adjust Brush Size**: Use the slider to make your brush bigger or smaller
3. **Magic Wand**: Click on the magic buttons to activate special effects:
   - 🌈 Rainbow Trail: Your brush changes colors as you draw
   - ✨ Sparkle Brush: Adds extra sparkles to your drawings
   - 🌟 Glow Paint: Makes your brush glow with magical light
   - 🎭 Magic Eraser: Special effects for removing parts of your drawing
4. **Draw Away**: Click and drag on the canvas to create your masterpiece
5. **Clear Canvas**: Click the "✨ Clear Canvas" button to start fresh

## 🛠️ Technical Details

- **Framework**: React 18 with TypeScript
- **Build Tool**: Vite for fast development and building
- **Styling**: CSS-in-JS with custom animations
- **Canvas**: HTML5 Canvas API for drawing functionality
- **Responsive**: Works on desktop, tablet, and mobile devices

## 🎨 Customization

The app is designed to be easily customizable:
- Add new colors by modifying the `colors` array in `App.tsx`
- Create new magical effects in the `MagicWand.tsx` component
- Adjust canvas size, brush properties, and UI elements as needed

## 🌈 Magic Behind the Scenes

PlaySlate uses several delightful features to create a magical experience:
- Particle system for sparkle effects
- Real-time animation loops for smooth interactions
- Touch event handling for multi-device support
- Gradient backgrounds and CSS animations
- Responsive design that works on any screen size

## 🔐 Backend & Authentication

The backend is built with Go and includes:
- User authentication system
- Google OAuth integration
- Magic link email authentication
- User subscription management
- Secure middleware for API protection
